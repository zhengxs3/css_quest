Introduction to HTML & CSS

What is HTML?

HTML stands for HyperText Markup Language. It is the structure of every webpage.

HTML decides:

  • What text appears
  • Where images go
  • What buttons or links exist
  • How the page is organized

Think of HTML as the “bones” of a website — everything you see on a page exists because HTML creates it.

Example 1 – Heading (<h2>)

Code:
<h2>This is a section title</h2>

This is a section title

What does it do?
  • <h1> = Main title
  • <h2> → <h6> = subtitles
  • Used for structure, not visual styling

Example 2 – Paragraph (<p>)

Code:
<p>This is a simple text paragraph.</p>

This is a simple text paragraph.

What does it do?
  • Creates a text block
  • Browser automatically adds spacing

Example 3 – Link (<a>)

Code:
<a href="https://example.com">Visit website</a>
Visit website
Key points:
  • href contains the target URL
  • Used to link pages, files, email, phone

Example 4 – Button (<button>)

Code:
<button>Click me</button>
Key points:
  • A clickable button
  • Often used with JavaScript

Example 5 – Image (<img>)

Code:
<img src="photo.jpg" alt="A photo" width="120">
Key points:
  • src = image source
  • alt = description text
  • Image tag is self-closing

Example 6 – List (<ul> + <li>)

Code:
<ul> <li>Apple</li> <li>Banana</li> </ul>
  • Apple
  • Banana
Key points:
  • <ul> = unordered list (bullets)
  • <li> = list item

What is CSS?

CSS stands for Cascading Style Sheets. If HTML is the “bones”, then CSS is the “skin and clothing”.

CSS controls the appearance of the webpage, including:

  • Colors
  • Sizes
  • Layouts
  • Animations
  • Borders, rounded corners

Without CSS, every webpage looks plain and unstyled. CSS makes websites beautiful, colorful, and modern.

Simple CSS examples:
  • Make a box blue
  • Create a circle using border-radius: 50%
  • Add rounded corners
  • Add spacing and shadows
Example 1 – Simple line

We draw a thin horizontal line using the border property.

width: 250px;
height: 0;
border-top: 3px solid black;
          
width: 200px;
height: 0;
border-top: 5px dashed blue;
          
width: 180px;
height: 0;
border-top: 4px dotted green;
          
width: 250px;
height: 0;
border-top: 6px double purple;
          
width: 220px;
height: 0;
border-top: 5px groove orange;
          
width: 230px;
height: 0;
border-top: 5px ridge brown;
          
width: 240px;
height: 0;
border-top: 5px inset gray;
          
width: 260px;
height: 0;
border-top: 5px outset red;

What do these properties mean?

  • width: the horizontal length of the line (you can use 260px, 200px, 150px, 100px…).
  • height: set to 0 here because we only want a line, not a box.
  • border-top: draws a line on the top edge. It has 3 parts: thickness style color, for example 3px solid black.

Common border styles:

  • solid – normal continuous line
  • dashed – broken line made of short dashes
  • dotted – line made of dots
  • double – two parallel lines
  • groove – looks like a 3D carved line
  • ridge – the opposite of groove, looks raised
  • inset – looks pressed into the page
  • outset – looks like it is coming out of the page

Extra: border-radius is used to round corners. It does not affect these lines, but it is very useful for boxes and buttons in the next examples.

Example 2 – Blue rectangle

Adding width, height and background creates a simple box.

width: 150px;
height: 80px;
background: #1CCAE8;

What do these properties mean?

  • width: how wide the box is.
  • height: how tall the box is.
  • background: the fill color of the box.
  • This creates a rectangle because width ≠ height.
Example 3 – Rounded rectangle

We add border-radius to round the corners.

width: 150px;
height: 80px;
background: #56D9BA;
border-radius: 15px;

What do these properties mean?

  • width: the width of the box.
  • height: the height of the box.
  • background: background color.
  • border-radius: 15px: rounds each corner by 15 pixels.

This shape is still a rectangle, but with smooth corners.

Example 4 – Circle

A perfect circle is made using equal width + height and border-radius: 50%.

width: 100px;
height: 100px;
background: #91E413;
border-radius: 50%;

What do these properties mean?

  • width & height: must be equal to form a circle.
  • background: fill color of the circle.
  • border-radius: 50%: transforms a square into a perfect circle.
Example 5 – Red box with border

This example shows how to style a box with rounded corners and a border.

width: 120px;
height: 80px;
background: red;
border-radius: 10px;
border: 3px solid black;

What do these properties mean?

  • width: how wide the box is.
  • height: how tall the box is.
  • background: the fill color of the box.
  • border-radius: 10px: rounds the corners by 10 pixels.
  • border: adds a black outline around the box.

px vs % for border-radius:

  • px = fixed corner size. Example: 10px → always the same curve.
  • % = based on the element’s size. Example: 50% → makes circles when width = height.
  • This box uses 10px because we want rounded corners, not a circle.

In short:
• 10px = rounded rectangle
• 50% = circle (when width = height)

How HTML and CSS Work Together

A real webpage uses both HTML and CSS together.

HTML creates the content (text, buttons, images). CSS makes the content look good (colors, size, style).

Example:

  • HTML creates a button
  • CSS makes the button green, bigger, and rounded

When combined, they form what you see on modern websites every day.

This section explains:
  • Why HTML needs CSS
  • Why CSS needs HTML
  • How they create real user interfaces

Example 1 – Colored Box

Code:
<div class="color-box"></div> <style> .color-box { width: 120px; height: 80px; background: #3498db; border-radius: 10px; } </style>

HTML:

  • <div class="color-box"> creates an empty box element.

CSS (what each line does):

  • width: 120px; → sets the box width.
  • height: 80px; → sets the box height.
  • background: #3498db; → gives the box a blue background.
  • border-radius: 10px; → rounds the corners (soft rectangle).

This is the simplest example of HTML + CSS: HTML = box, CSS = shape + color.

Example 2 – Styled Button

Code:
<button class="btn-green">Click me</button> <style> .btn-green { background: #28c76f; color: white; padding: 10px 20px; border-radius: 8px; border: none; font-size: 16px; cursor: pointer; font-weight: bold; } </style>

HTML:

  • <button>Click me</button> creates a clickable button.
  • class="btn-green" links the button to the CSS rules.

CSS (focus on style):

  • background: #28c76f; → green background color.
  • color: white; → white text on the button.
  • padding: 10px 20px; → makes the button bigger (inner spacing).
  • border-radius: 8px; → rounded corners, modern look.
  • border: none; → removes the default browser border.
  • font-size: 16px; → sets readable text size.
  • font-weight: bold; → makes the text bold.
  • cursor: pointer; → shows a hand cursor on hover (looks clickable).

Here, HTML gives a button; CSS turns it into a visually appealing call-to-action.

Example 3 – Card Component

Code:
<div class="card"> <h4>Hello!</h4> <p>This is a simple card.</p> </div> <style> .card { width: 200px; padding: 15px; border-radius: 10px; background: #ff7979; box-shadow: 0 2px 8px rgba(0,0,0,0.1); text-align: center; } </style>

Hello!

This is a simple card.

HTML structure:

  • <div class="card"> – the container of the card.
  • <h4> – the card title.
  • <p> – the card text.

CSS (how it becomes a real UI card):

  • width: 200px; → sets a fixed card width.
  • padding: 15px; → adds space inside the card so text doesn’t touch the edges.
  • border-radius: 10px; → rounds the corners for a softer, modern look.
  • background: #ff7979; → the background color of the card.
  • box-shadow: 0 2px 8px rgba(0,0,0,0.1); → adds a soft shadow to make the card “float”.
    • 0 → horizontal offset (shadow stays centered).
    • 2px → vertical offset (shadow moves slightly downward).
    • 8px → blur radius (bigger value = softer shadow).
    • rgba(0,0,0,0.1) → shadow color (black with 10% opacity).
  • text-align: center; → centers the title and text horizontally.

In short: HTML provides structure, and CSS adds style, color, spacing, and shadow to transform it into a clean UI card.
